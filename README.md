### Description
A full-stack JS app with ES6/ES2015 React.js featuring universal Redux, React Router, Hot reloading, CSS modules, Express 4.x, and MongoDB.


### devUsage

###### Connect to MongoDB

```
mongod

```
###### Install dependencies and run dev
```
npm install
npm run dev

```
